# #1-1. Image Generation Workflow

이번 실습에서는 Image Generation Workflow를 직접 구축해보고 실행시켜보는 실습을 진행하겠습니다.



가장 기본이 되는 모델인 Stable Diffusion Workflow를 실행해보고 ComfyUI 사용법을 익히는 시간이 되었으면 합니다.



Stable Diffusion은 여러 모델이 합쳐진 Multimodal Model입니다.

<figure><img src="../.gitbook/assets/image (23).png" alt=""><figcaption></figcaption></figure>

기본적인 모델의 구조는 위 사진과 같이 구성되어 있으며, 각 모델 요소는 아래와 같습니다.

* Variational Autoencoder (VAE) : 이미지를 픽셀 공간에서 더 작은 Latent space로 압축하여 이미지의 더 근본적인 의미론적 의미를 포착함
* U-Net : ResNet backbone으로 구성된 이 블록은 순방향 확산에서 출력된 결과를 역방향으로 노이즈 제거를 수행하여 잠재 표현을 얻는다. 다운 블록, 업 블록, 미드 블록으로 구성되어 U자 형태의 네트워크 아키텍처를 형성함
* Text-Encoder : 텍스트 조건부 생성을 위해 고정된 사전 훈련된 CLIP ViT-L/14 텍스트 인코더를 사용하여 텍스트 프롬프트 임베딩 공간으로 변환



### Stable Diffusion 작동 순서



**1. 텍스트 입력 및 인코딩 단계**

* **텍스트 프롬프트 입력**: 사용자가 "a cat sitting on a chair"와 같은 텍스트 프롬프트를 입력한다.
* **텍스트 인코딩**: CLIP ViT-L/14 텍스트 인코더가 이 텍스트를 벡터 형태의 임베딩으로 변환한다.
* **임베딩 전달**: 생성된 텍스트 임베딩은 U-Net의 교차 주의(cross-attention) 레이어에 전달되어 이미지 생성 과정을 조건화한다.

**2. 초기 노이즈 생성 단계**

* **가우시안 노이즈 생성**: 무작위 가우시안 노이즈가 생성된다.
* **잠재 공간 초기화**: 이 노이즈는 잠재 공간에서의 초기 상태로 사용된다. 이미지-이미지 생성의 경우, 기존 이미지의 잠재 표현에 노이즈가 추가된다.
* **노이즈 수준**: 텍스트-이미지 생성에서는 완전한 노이즈(t=T)로 시작하고, 이미지-이미지 생성에서는 Denoising Strength 파라미터에 따라 노이즈 양이 결정된다.

**3. 노이즈 스케줄 설정 단계**

* **스케줄러 설정**: 스케줄러는 각 타임스텝에서의 노이즈 수준을 정의한다.
* **샘플러에 전달**: 이 노이즈 스케줄은 샘플러에게 전달되어 각 단계에서 얼마나 노이즈를 제거할지 결정한다.
* **스케줄 유형**: 선형, 코사인, 지수 등 다양한 스케줄이 사용될 수 있으며, 각각 노이즈 감소 패턴이 다르다.

**4. 반복적 샘플링 단계**

* **노이즈 예측 단계**:
  * 현재 노이즈가 있는 잠재 벡터가 U-Net에 입력된다.
  * U-Net은 텍스트 임베딩을 조건으로 사용하여 잠재 벡터에 있는 노이즈를 예측한다.
  * 이 예측된 노이즈는 샘플러에 전달된다.
* **노이즈 제거 단계**:
  * 샘플러는 예측된 노이즈를 처리하고 이를 스케줄러에 전달한다.
  * 스케줄러는 현재 타임스텝에 맞는 노이즈 제거 방식을 적용한다.
  * 노이즈가 제거된 새로운 잠재 벡터가 생성된다.
* **반복 과정**:
  * 이 과정은 설정된 샘플링 스텝 수(예: 20, 50)만큼 반복된다.
  * 각 반복마다 잠재 벡터는 점점 더 정제되어 원하는 이미지에 가까워진다.
  * 타임스텝은 T에서 시작하여 각 단계마다 감소하여 최종적으로 0에 도달한다.

**5. 최종 이미지 생성 단계**

* **잠재 벡터 디코딩**: 최종적으로 정제된 잠재 벡터는 VAE 디코더로 전달된다
* **픽셀 공간 변환**: VAE 디코더는 이 잠재 벡터를 픽셀 공간으로 변환하여 최종 이미지를 생성한다.
* **이미지 출력**: 생성된 이미지는 사용자에게 표시된다.



### Image Generation Workflow 만들기



1. 처음 ComfyUI에 로그인하면 빈 캔버스를 볼 수 있습니다.

<figure><img src="../.gitbook/assets/image (21) (1).png" alt=""><figcaption></figcaption></figure>



2. 빈 캔버스에 더블클릭 하여 찾는 노드를 검색하거나 좌측 메뉴 줄의 노드를 클릭하여 드래그하면 사용하고자 하는 노드를 캔버스에 배치할 수 있습니다.

<figure><img src="../.gitbook/assets/image (24).png" alt=""><figcaption></figcaption></figure>



3. 먼저 체크포인트 로드 노드를 검색하여 캔버스에 배치합니다.

4. 프롬프트를 입력하여 CLIP 인코딩을 위한 **CLIP 텍스트 인코딩(프롬프트)** 노드를 검색하여 2개를 배치합니다.
   1. 한개의 노드를 만들고 클릭 후 복사(CTRL+C), 붙여넣기하면 노드를 복사할 수 있습니다.

5. 초기 latent image를 만들기 위해 빈 잠재 이미지 노드를 검색하여 배치합니다.

샘플러 이름 : euler
