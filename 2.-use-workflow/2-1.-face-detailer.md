# #2-1. Face Detailer

이번 실습은 Face Detailer를 이용한 이미지 수정 실습을 진행해보겠습니다. 생성형 AI로 이미지를 생성할 때 마음에 들지 않는 부분이나 특정 부분만 퀄리티를 높이고 싶은 경험을 해보신 적이 많은데 이러한 경우 Inpainting 기능을 활용하여 특정 부분을 마스크 처리하여 지운 뒤 이미지를 덧씌우는 것과 비슷한 작업을 수행하게 됩니다.



이때 사용할 수 있는 노드로 Face Detailer는 특정 부분을 자동으로 식별 및 추출 후 확대한 뒤 수정하는 기능이라고 할 수 있습니다.
