# #2-2. Video Generation Workflow

이번 시간에는 Video Generation Workflow 템플릿을 활용하여 Text-to-Video / Image-to-Video 실습을 진행하겠습니다.



Text-to-Video는 말 그대로 입력한 텍스트 프롬프트를 이용하여 비디오를 생성하는 기법이며, Image-to-Video는 입력한 Reference Image와 텍스트 프롬프트를 이용하여 비디오를 생성하는 기법입니다.  이번 실습에서는 Wan 2.1 을 사용하여 텍스트 프롬프트를 입력받아 ComfyUI에서 비디오를 생성하는 실습과 Framepack을 사용하여 이미지 프롬프트를 입력받아 비디오를 생성하는 실습을 진행해 보겠습니다.



Wan2.1은 Apache 2.0 License로 제공되며, 14B(140억 파라미터) / 1.3B(13억 파라미터) 두 가지 버전으로 제공됩니다. 또한 커뮤니티에서 GGUF 및 양자화 버전을 제작하여 효율적으로 활용할 수 있습니다.



Framepack은 ControlNet의 저자인 Lvmin Zhang 박사 팀이 개발한 AI 비디오 생성 기술입니다. 동적 컨텍스트 압축, 드리프트 방지 샘플링, 감소된 VRAM 요구량, 오픈소스 통합 등의 특징을 가지며 현재 텐센트의 Hunyuan video 모델에 통합되어 있습니다.
