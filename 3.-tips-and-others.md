### 소소한 팁

* Reroute node
  * 단순 노드를 연결하는 역할
  * 선이 복잡하거나 너무 멀리서 끌어다 쓸 때 사용하는 연장선
* Workflow 노드 연결선 변경
  * 기본은 부드러운 곡선으로 설정되어 있음
  * 각지게 또는 직선으로 변경 가능
* 노드 복사는 CTRL+C / 노드 붙여넣기 CTRL+V
* 커뮤니티에는 이미지로 워크플로우를 공유하는 (webp) 경우가 있음, 이미지를 다운로드 받아서 드래그하면 워크플로우가 자동으로 생성됨
